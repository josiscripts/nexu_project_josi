# Nexu Project Josi

Futuristic, minimalist, and professional social network for students.

## Features
- Modern UI/UX with Vite and React
- Tailwind CSS styling
- Framer Motion animations
- Professional design aesthetics

## Getting Started

1. Clone the repository
2. Install dependencies: `npm install`
3. Run development server: `npm run dev`

## Project Structure
- `src/`: Source code
- `public/`: Static assets
- `onboardings_nexu.pdf`: Project documentation/onboarding guide
